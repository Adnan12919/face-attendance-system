FACE REGISTRATION FOLDER
========================

teacher/
  • Add exactly ONE photo of the teacher.
  • Filename: teacher.jpg (or .png / .jpeg)

students/
  • Add ONE photo per student.
  • Name each file after the student ID:
      2024457224_Rishabh.jpg
      2024126451_Adnan.jpg
  • The filename (without extension) becomes the
    attendance record ID.

Supported formats: .jpg  .jpeg  .png  .bmp  .webp
Each photo must contain exactly ONE clearly visible face.
